# Guides

Topics that don't fit into getting-started, architecture, operations,
or security — usually longer-form walkthroughs for optional upgrades
or side quests.

- [Upgrading to hydration](hydration-upgrade.md) — path from the
  current SSR-only build to a full Leptos `hydrate` mode where the
  client side becomes reactive. Requires `wasm32-unknown-unknown`.

As new optional capabilities land — the PostgreSQL backend, OIDC,
the AI assistant — each one gets a guide here explaining how to
opt in, what trade-offs come with it, and which parts of the
workspace are affected.
